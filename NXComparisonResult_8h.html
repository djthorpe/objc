enum NXComparisonResult

Enumeration representing the result of a comparison operation.

NXComparisonResult is used throughout the NXFoundation framework to represent the outcome of comparison operations between objects. The values are designed to be compatible with standard C library comparison functions and can be used directly in sorting algorithms.
