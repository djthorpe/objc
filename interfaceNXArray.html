◆ remove:()

- (BOOL) remove: (id< RetainProtocol >)  object

Removes the first occurrence of the specified object from the array.

Parameters
objectThe object to remove from the array.
Returns
YES if the object was found and successfully removed, NO otherwise.

This method searches the array from the beginning to find the first occurrence of an object that is equal to the specified object (using the isEqual: method). If found, the object is removed and all subsequent elements are shifted down to fill the gap. The array's count is decremented by one. If the object is not found in the array, the method returns NO and the array remains unchanged.

◆ removeAllObjects()

- (void) removeAllObjects

Removes all objects from the array.

This method clears the array by releasing all objects and resetting the internal data structure. The array's count is set to zero and its capacity remains unchanged.
