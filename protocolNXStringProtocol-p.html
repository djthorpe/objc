Detailed Description

Protocol for string-like objects.

The NXStringProtocol defines the minimal interface that string-like objects must implement to provide basic string functionality. This enables different string implementations to be used polymorphically throughout the framework.

Classes conforming to this protocol can be used wherever basic string operations such as accessing the C string representation or getting the length are required.
