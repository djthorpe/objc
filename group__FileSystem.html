Cleans up the file system volume.

◆ fs_vol_init_file()

fs_volume_t* fs_vol_init_file ( const char *  path,
size_t  size 
)

Initializes a filesystem-based volume.

Open an existing, or create a new filesystem-based volume with the specified size. The actual size will be equal or greater than the requested size. If the volume already exists, an attempt will be made to extend the filesystem to the requested size.

If files are not supported on the platform, this function will return NULL.

◆ fs_vol_init_memory()

fs_volume_t* fs_vol_init_memory ( size_t  size)

Initializes a volatile memory-backed file system volume.

Create a new in-memory file system volume with the specified size. The actual size will be equal or greater than the requested size. If the volume is already initialized, it will be finalized and replaced.

Reads a directory and returns the next file entry.

◆ fs_vol_size()

size_t fs_vol_size ( fs_volume_t *  volume)

Determines if the file system volume is valid and usable.

This function checks if the provided volume handle is initialized and ready for file operations. It returns the size of the volume in bytes if valid, or zero if invalid.
